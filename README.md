## How to Build
1. To build it you'll need [Python](https://www.python.org/downloads/), [CMake](https://cmake.org/install/), and some other libraries.

2. Install libraries with:
    `python -m pip install -r requirements.txt`

3. Build:
    - Linux/Mac:
        ```bash
        > make html
        ```
    - Windows:
        ```bat
        > make.bat html
        ```
